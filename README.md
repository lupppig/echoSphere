#  Echo Sphere

A ball bounces inside a circle. That's it. That's the whole thing.

I was just fucking around with [Ebitengine](https://ebitengine.org/) and [beep](https://github.com/faiface/beep) and I produced this garbae lol lmaooooooooooooooo day in the life of an unemployed Software Engineer MF


## Run It

```bash
go run .
```

That's it. No config. No flags. No purpose.

## Dependencies

| Package | Why |
|---|---|
| [ebitengine](https://github.com/hajimehoshi/ebiten) | Drawing circles and lines at 60fps like a normal person |
| [beep](https://github.com/faiface/beep) | Making sine waves pretend to be music |
